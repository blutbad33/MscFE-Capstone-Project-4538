# Contributing to Cryptocurrency Short-Term Trading Model

We welcome contributions to the Cryptocurrency Short-Term Trading Model project! If you would like to contribute, please follow these guidelines.

## How to Contribute

1. **Feel free to fork the Repository**: Start by forking the repository on GitHub.

2. **Clone Your Fork and add your own ideas**: Clone your fork to your local machine:

   ```bash
   git clone https://github.com/blutbad33/MscFE-Capstone-Project-4538/new/main.git
   cd cryptocurrency-trading-model
   ```

3. **Create a New Branch**: Create a new branch for your feature or fix:

   ```bash
   git checkout -b feature/my-new-feature
   ```

4. **Make Your Changes**: Implement your changes, adhering to the existing coding style.

5. **Commit Your Changes**: Commit your changes with clear and descriptive commit messages.

6. **Push to the Branch**: Push your changes to your GitHub repository:

   ```bash
   git push origin feature/my-new-feature
   ```

7. **Submit a Pull Request**: Go to the original project on GitHub and submit a pull request.

## Code of Conduct

Our project adheres to a Code of Conduct that all contributors are expected to follow. This ensures a welcoming and inclusive environment for everyone.

## Coding Standards

- Write clear, understandable, and commented code.
- Follow the existing code style and conventions.
- Ensure that your code has been tested thoroughly.

## Reporting Bugs

When reporting bugs:

- Use the GitHub Issues page.
- Include a clear description of the problem.
- Provide additional information like screenshots or error messages.

## Feature Requests

We also welcome feature requests. Please:

- Use the GitHub Issues page to submit feature requests.
- Clearly describe the proposed feature and its benefits.

## Questions and Support

For any questions or support, please use the GitHub Issues page. We will do our best to assist you.

## Acknowledgements

Thank you for considering contributing to the Cryptocurrency Short-Term Trading Model. Your efforts and contributions are what make the open-source community a wonderful place to learn, inspire, and create.
